This module wires up the [Shadcn/UI Table](https://ui.shadcn.com/docs/components/table) components with Laravel's default paginator. Two examples are provided, the first uses inertia as the data source and the second loads data as json via fetch.

## Inertia with Numeric Pagination
This example loads it's data via Inertia requests and shows numbered page controls. As you paginate this table the page number is reflected in the address bar.

## Installation Steps

This module is designed to be installed into applications derived from the official Laravel React starter kit. For more information please read the <a href="/docs/introduction" className="underline">Introduction page</a>.

### 1: Install the Module

You can install the table module using the shadcn registry system:

```bash
npx shadcn@latest add https://astrify.github.io/table.json
```

In addition to installing the necessary JS dependencies, this command installs the following files into your application:

```bash
app/
└── Http/
    └── Controllers/
        └── Astrify/
            └── TableController.php
resources/
└── js/
    └── pages/
        └── table.tsx
```

### 2: Configure Routes

Add the following route to your `routes/web.php` file:

```php
use App\Http\Controllers\Astrify\TableController;

Route::middleware(['auth'])->group(function () {
    Route::get('/table', [TableController::class, 'index'])->name('table.index');
});
```

### 3: Seed Sample Data (Optional)

To quickly test the table component with sample data, you can seed your database with users:

```bash
php artisan tinker
User::factory()->count(50)->create();
```

Finally, navigate to the `/table` route in your Laravel app to test the module.

## Usage

### Basic Implementation (Numeric Pagination)

```tsx
import { router } from '@inertiajs/react';
import PaginatedTable from '@/components/astrify/table/paginated-table';

interface User {
  id: number;
  name: string;
  email: string;
  created_at: string;
}

interface Props {
  users: {
    data: User[];
    current_page: number;
    last_page: number;
    total: number;
    links: Array<{
      url: string | null;
      label: string;
      active: boolean;
    }>;
  };
}

export default function UsersPage({ users }: Props) {
  const handlePageChange = (url: string | null) => {
    if (!url) return;
    router.get(url, {}, { preserveScroll: true, preserveState: true });
  };

  return (
    <div className="container mx-auto py-8">
      <h1 className="text-2xl font-bold mb-6">Users</h1>

      <PaginatedTable
        columns={['ID', 'Name', 'Email', 'Created']}
        data={users.data.map((user) => [
          user.id,
          user.name,
          user.email,
          user.created_at,
        ])}
        pagination={{
          type: "numeric",
          currentPage: users.current_page,
          lastPage: users.last_page,
          links: users.links,
          onChangePage: handlePageChange,
          total: users.total,
        }}
      />
    </div>
  );
}
```

## Backend Implementation

The table components work seamlessly with Laravel's pagination. Here's a typical controller implementation:

```php
<?php

namespace App\Http\Controllers;

use App\Models\User;
use Inertia\Inertia;
use Inertia\Response;

class UsersController extends Controller
{
    public function index(): Response
    {
        $users = User::select('id', 'name', 'email', 'created_at')
            ->orderBy('id', 'desc')
            ->paginate(10);

        return Inertia::render('users/index', [
            'users' => $users,
        ]);
    }
}
```

Laravel's paginator automatically provides all the necessary pagination metadata including:
- `data`: Array of current page results
- `current_page`: Current page number
- `last_page`: Total number of pages
- `total`: Total number of records
- `links`: Array of pagination links

## Component Props

### PaginatedTable

| Prop | Type | Description |
|------|------|-------------|
| `columns` | `string[]` | Array of column header labels |
| `data` | `(string \| number)[][]` | 2D array of table data rows |
| `pagination.type` | `"simple" \| "numeric"` | The pagination style to use |
| `pagination.currentPage` | `number` | Current page number |
| `pagination.lastPage` | `number` | Total number of pages |
| `pagination.links` | `PaginationLinkType[]` | Array of pagination link objects from Laravel |
| `pagination.onChangePage` | `(url: string \| null) => void` | Callback function when page changes |
| `pagination.total` | `number` (optional) | Total number of records |
